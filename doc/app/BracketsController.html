    class BracketsController

    This controller displays the available Bracket Groupings and allows the user to select one for updating the Bracketcontests contained within. Brackets themselves are not created nor destroyed nor updated by this controller, it merely serves as a routing device. Therefore, it ha only two actions: index and show.
